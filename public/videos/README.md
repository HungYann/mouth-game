# 视频学习素材目录

当玩家失败时，系统会按关卡顺序播放这里配置的视频，玩家看完后才能继续重试。

## 使用方法

1. 把学习视频放到本目录，例如：
   - `1.mp4`
   - `lesson-2.mp4`
   - `lesson-3.mp4`
2. 修改 `manifest.json`，保持为数组格式，路径从站点根路径开始（数组顺序即关卡顺序）：

```json
[
  "/videos/1.mp4",
  "/videos/2.mp4",
  "/videos/3.mp4"
]
```

## 注意

- 推荐使用 `mp4`（H.264 编码）以保证浏览器兼容性。
- 如果 `manifest.json` 中的文件不存在，游戏会提示视频不可用，无法继续重试。
- 若关卡数超过视频数，会自动使用最后一个视频作为兜底。
